# Kayak Travel Platform - Admin Dashboard

A comprehensive distributed travel booking platform with microservices architecture.

## Features
- Admin Dashboard with Analytics
- User Management
- Billing & Revenue Tracking
- Redis Caching
- Docker & Kubernetes Deployment

## Tech Stack
- **Frontend:** React, Material UI, Recharts
- **Backend:** Node.js, Express
- **Databases:** MySQL, MongoDB
- **Cache:** Redis
- **Message Queue:** Kafka
- **Deployment:** Docker, Kubernetes

## Quick Start
```bash
npm install
npm start
```
